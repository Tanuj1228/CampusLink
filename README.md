## Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **ORM:** Sequelize
* **Real-time:** Socket.io
* **API Layer:** GraphQL, REST (Auth)
* **Frontend:** HTML, CSS, Vanilla JS

## Setup Instructions
1. Run `npm install`.
2. Create a MySQL database named `job_portal_db`.
3. Configure `.env` with DB credentials and JWT secret.
4. Run `npm run dev` to start the server on port 3000.

## Usage
1. Open `http://localhost:3000/auth.html`
2. Register/Login as a Company to access the Company Dashboard and post jobs.
3. Register/Login as a Student to view jobs, receive real-time alerts, and apply.
